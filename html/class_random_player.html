Detailed Description

PlayerBase-bõl leszármazott osztály. A heterogén kollekció része. A randomizált játékost valósítja meg, a cstdlib library-ban található rand() függvény segítségével. az srand() meg-hívásáért nem felel, a példányosítása elõtt biztosítani kell a lefutását.
